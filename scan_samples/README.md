# scan_samples フォルダについて（教員が用意するもの）

このフォルダには、**実際に紙へ印刷して手書きで記入し、スキャン／撮影した解答用紙の画像**を置く。
ダミーで作った画像（きれいすぎる・歪みがない・文字が印刷フォントそのまま等）では、
このシステムの一番の見せ場である「スキャン画像の位置合わせ補正 → AI文字認識 → 自動採点」の
リアルさが伝わらないため、ここだけは実物を用意してほしい。

## 用意してほしいもの

1. **`sample_gui.py`（週次小テストタブ）で `week1` の「小テストPDF生成」を実行**し、
   出力された `output/Sample_梁のたわみ計算（サンプル科目）/R8/01小テスト_片持ち梁+集中荷重.pdf` を
   1〜2部印刷する（別の学生を演じるなら複数部）。
2. 印刷したプリントに、学生になったつもりで手書きで記入する。
   - **出席番号欄**（左上、2マス）: 十の位・一の位を1マスに1桁ずつ書く（例: `01`）。
   - **各解答欄**（1マスに1文字、次の順で7マス）:
     `[符号 +/-] [1の位] . [小数第1位] [小数第2位]  ×10  [指数の符号 +/-] [指数の十の位] [指数の一の位]`
     例えば答えが `3.14×10⁴` なら `+` `3` `1` `4` … `+` `0` `4` と書く。
     - 正確な正答が知りたい場合は、`Exercises()`（演習PDF、模範解答つき）を同じ週で生成するか、
       Python で `week1.Question1(seed).answer` を直接呼んで確認できる（seed は
       `config.xlsx` の `date` を `YYYYMMDD` にした値）。
     - **デモの目的は「パイプラインが実際に動くこと」の確認なので、多少値が不正確でもよい**
       （0点になっても、認識結果が表示されて修正できることを確認できれば十分）。
   - **四隅（と右下付近）の黒塗りマーカーは絶対に隠したり上から書き込んだりしないこと**
     （スキャン画像の位置合わせに使うため）。
3. スマホのカメラ、またはスキャナーで撮影・スキャンする。
   - 多少の傾き・影・歪みが入っている方が「実運用に近いテスト」になってよい
     （むしろ完全に真正面から撮った歪みゼロの画像より実用上のテストとして適切）。
   - JPEG で保存し、このフォルダに `week1_01.jpg` のような名前で置く（複数枚あれば
     `week1_02.jpg` のように増やしてよい）。

## 用意してもらったあとの使い方

置いてもらった画像を、実際に採点対象になる場所へコピーする（初回は手動でよい）。

```
output/Sample_梁のたわみ計算（サンプル科目）/R8/minitest/1st/week01/scan/
```

へ `scan_samples/*.jpg` をコピーしてから、GUIの「採点」ボタンを押す。
