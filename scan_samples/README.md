# scan_samples フォルダについて

このフォルダには、実際に紙へ印刷して手書きで記入し、スキャン／撮影した解答用紙の画像を
同梱している。ダミーで作った画像（きれいすぎる・歪みがない・文字が印刷フォントそのまま等）
では「スキャン画像の位置合わせ補正 → AI文字認識 → 自動採点」というこのシステムの
一番の見せ場のリアルさが伝わらないため、実物を使っている。

## 解答欄フォーマット

- **出席番号欄**（左上、2マス）: 十の位・一の位を1マスに1桁ずつ書く（例: `01`）。
- **各解答欄**（1マスに1文字、次の順で7マス）:
  `[符号 +/-] [1の位] . [小数第1位] [小数第2位]  ×10  [指数の符号 +/-] [指数の十の位] [指数の一の位]`
  例えば答えが `3.14×10⁴` なら `+` `3` `1` `4` … `+` `0` `4` と書く。
- 四隅（と右下付近）の黒塗りマーカーはスキャン画像の位置合わせに使う。

## 使い方

GUIの「個別再テスト」タブで「QR一括採点（フォルダ自動振分）」ボタンを押すだけでよい。
採点前に `scan_samples/*.jpg` の未コピー分が自動的に

```
output/{科目}/{年度}/minitest/retry_scan/
```

へ取り込まれる（`packages/quiz/Runner.py` の `sync_scan_samples`）。手動コピーは不要。
